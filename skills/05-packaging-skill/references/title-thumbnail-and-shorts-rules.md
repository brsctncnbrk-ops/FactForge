# Title, Thumbnail, Shorts & A/B Rules

## Titles (English)

- Curiosity without deception: the video must fully deliver what the title implies (Promise Match — the human checklist judges this; make it easy to pass).
- Short and strong; front-load the tension word. Name the subject plainly — no riddle titles.
- Ban list: "In this video...", generic listicle phrasing, ALL-CAPS whole titles, year-bait ("...in 2026") unless factual.
- Patterns that work for this channel's tone:
  - Contrarian correction: "Why Rome Was Already Dead Before It Fell"
  - Blunt reframe: "Rome Didn't Fall. It Broke."
  - Superlative + puzzle: "Why the Richest Empire in History Collapsed"
  - Fatal property: "The Empire That Became Too Big to Survive"
- Numbers in titles require a VERIFIED fact (never SINGLE-SOURCE — patch Section 6).

## Thumbnails

- One glance, one idea: main conflict visible at 120×68 px.
- Text: 2–5 words, ALL CAPS, high contrast; must not repeat the title verbatim — complement it.
- Style-consistent with the video look: simple 2D infographic, cinematic mood, dark/neutral background, one strong focal element (bust, map, silhouette).
- No fabricated imagery of real events that implies false specifics (honesty rule extends to visuals).

## Shorts ideas (3+ per video)

Each idea must be producible from existing scene data with a 9:16 template variant — only `stat-card`, `text-emphasis`, `map-scene` have vertical variants (aspectRatio prop). Required fields per idea: Shorts Title, Clip Segment / Moment, Hook Line, Visual Idea (which 9:16 template + key props). If an idea needs any other template, it is not a valid Shorts idea for this system — replace it.

## A/B testing

- Change ONE variable per test (title OR thumbnail), state which is held fixed.
- Provide at least: one title A/B pair and one thumbnail A/B pair.
- Pick variants that test a hypothesis ("does the contrarian angle beat the superlative angle?"), not cosmetic shuffles.

## CTR Angle Explanation

One short paragraph: which curiosity gap the packaging opens, why it is honest (the video actually closes it), and what common belief it corrects. This doubles as the input for the Script Hook Promise — keep them consistent.

## Pinned comment & description

- Pinned comment: one engagement question tied to the topic, or the single most surprising verified fact as a teaser.
- Description first two lines carry the hook (visible before "more"); keep the misconception→reveal structure from the brief's format; Attribution Block goes at the bottom (05B).
